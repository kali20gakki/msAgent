# 量化自动调优背景知识

## 什么是modelslim？

ModelSlim 是华为昇腾提供的大模型量化工具套件，可以将 LLM 权重压缩至 INT4/INT8 等低位宽，降低显存占用并提升推理性能。适用于昇腾 NPU 上的大模型部署场景。
ModelSlim 已支持 MCP 集成，可通过 MCP 工具暴露量化、评测、history 管理等接口。

## 什么是量化自动调优？

量化是有损压缩，量化后模型在数据集测评时会有精度损失。**量化自动调优一套流程，用来寻找指定精度损失约束下的最优量化配置。**

**工作原理：**

量化自动调优通过调整量化算法、回退特定敏感层等手段，寻找最优量化配置使精度达标。

**核心优势：**
- 免手工调参：自动决定保留哪些层为高精度、调整量化方案
- 闭环反馈：基于实测指标而非经验值
- 最优收敛：多轮迭代找最佳压缩比与精度平衡
